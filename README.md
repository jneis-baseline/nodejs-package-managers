# npm-docs
further exploring npm

### npm cli cheatsheet

* list config
> npm config ls

* project setup
> npm init

* deps setup
> npm install <dep> --save
> npm install <dep> --save-dev
> npm uninstall <dep> --save

* deps listing
> npm ls

* deps cleanup
> npm prune

* deps version mgmt
> npm outdated
> npm update
